# Toán Rời Rạc (Discrete Mathematics)

## 📌 Giới thiệu
Dự án này lưu trữ mã nguồn, bài tập và các thuật toán thực hành liên quan đến môn học **Toán rời rạc**. Đây là không gian để cài đặt và ôn tập các kiến thức nền tảng quan trọng trong khoa học máy tính, từ cấu trúc rời rạc, lý thuyết đồ thị đến các thuật toán tối ưu.

## 🚀 Các chủ đề chính
*(Lưu ý: Tùy chỉnh danh sách này dựa trên các bài tập bạn đã code trong repo)*

* **Lý thuyết đồ thị (Graph Theory):** * Cài đặt các thuật toán duyệt đồ thị: BFS, DFS.
    * Tìm đường đi ngắn nhất: Dijkstra, Floyd-Warshall, Bellman-Ford.
    * Bài toán cây khung nhỏ nhất: thuật toán Prim, Kruskal.
* **Thuật toán sinh & Quay lui (Combinatorics & Backtracking):**
    * Sinh chuỗi nhị phân, sinh tổ hợp, sinh hoán vị.
    * Giải quyết các bài toán duyệt toàn bộ (như bài toán N-Hậu, bài toán người du lịch TSP).
* **Logic & Đại số Boole:** Các bài toán xử lý hàm logic, bảng chân lý.
